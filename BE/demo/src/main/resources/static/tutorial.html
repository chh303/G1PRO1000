Efficient Algorithms

Efficient algorithms reduce runtime and power consumption. A good algorithm can be the difference between milliseconds and minutes.

Comparison of two algorithms:

// Poor solution:
for (int i = 0; i < n; i++) {
  for (int j = 0; j < n; j++) {
    if (data[i] == data[j]) {}
  }
}

// Better solution using HashSet:
Set<Integer> set = new HashSet<>();
for (int val : data) {
  if (set.contains(val)) {}
  else set.add(val);
}

Quiz: What is the advantage of using a HashSet in the example above?

It reduces complexity from O(n²) to O(n), and uses less CPU.

Optimized Code

Optimization is about more than speed. It's also about using fewer resources and reducing environmental impact.

Example of unnecessarily complex code:

// Not optimized:
String result = "";
for (String s : list) {
  result += s;
}

// Better solution using StringBuilder:
StringBuilder sb = new StringBuilder();
for (String s : list) {
  sb.append(s);
}
String result = sb.toString();

Sustainable Web Design

Sustainable web design is about reducing data load and power usage. Use optimal image compression, lazy loading, and a simple layout.

  • Avoid large videos on the front page.
  • Use system fonts instead of loading new ones.
  • Use dark mode where appropriate.
